## Requisitos

- Python 3.8 o superior

## Archivos

| Archivo | Descripción |
|---|---|
| `ecoflor.py` | Código fuente principal |
| `cosechas.txt` | Registros de cosechas |
| `incidencias.txt` | Registros de incidencias |

## Ejecución

Coloca los tres archivos en la misma carpeta y ejecuta:

```
python ecoflor.py
```

## Menú

| Opción | Descripción |
|---|---|
| 1 | Registrar cosecha |
| 2 | Ver historial de cosechas |
| 3 | Registrar incidencia |
| 4 | Ver incidencias |
| 5 | Análisis integral |
| 6 | Salir |
